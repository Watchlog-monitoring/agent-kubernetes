Watchlog Agent is running as a DaemonSet on all nodes.

To check the status:
  kubectl get pods -n {{ .Values.namespace.name }} -l app.kubernetes.io/name={{ include "watchlog-agent.name" . }}

To view agent logs:
  kubectl logs -n {{ .Values.namespace.name }} -l app.kubernetes.io/name={{ include "watchlog-agent.name" . }} -c agent

To view vector logs:
  kubectl logs -n {{ .Values.namespace.name }} -l app.kubernetes.io/name={{ include "watchlog-agent.name" . }} -c vector
